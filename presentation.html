Stage 2 - Notebook Cell 3

üß† MACHINE LEARNING: CNN Architecture

Model Architecture (InspectorCNN):

  • Input Layer: 128√ó128 RGB images (3 channels)
  • Feature Extraction: 3 convolutional blocks with ReLU activation and MaxPool downsampling
  • Block 1: Conv2d(3‚Üí32) + ReLU + MaxPool(2√ó2)
  • Block 2: Conv2d(32‚Üí64) + ReLU + MaxPool(2√ó2)
  • Block 3: Conv2d(64‚Üí128) + ReLU + MaxPool(2√ó2)
  • Classifier: Flatten ‚Üí Dense(2048‚Üí512) + Dropout(0.5) ‚Üí Output(6 classes)

Training Configuration:

  • Image size: 128√ó128 pixels
  • Batch size: 64 images
  • Epochs: 5
  • Optimizer: Adam with learning rate 0.001
  • Loss function: CrossEntropyLoss
  • Device: CUDA (GPU) or CPU fallback

How It Works:

Feature Extraction:
Convolutional layers automatically learn visual features like edges, textures, colors, and patterns that distinguish fresh fruit from rotten fruit. Each block reduces spatial dimensions while increasing feature depth.
Classification:
After feature extraction, the flattened feature vector passes through fully-connected layers. Dropout (0.5) prevents overfitting by randomly deactivating neurons during training.
Output:
6 output neurons represent probabilities for each class:
‚Ä¢ freshapples, rottenapples
‚Ä¢ freshbanana, rottenbanana
‚Ä¢ freshoranges, rottenoranges
